## Decisions made

- **Bundle uses stdlib only** — no new dependencies (html.parser, tempfile, zipfile, shutil). Keeps the project lightweight.
- **Log format is tab-separated** — simple to parse, human-readable, no need for a database.
- **Path rewriting uses os.path.relpath** — reliable cross-platform relative path computation.
- **try/finally for temp cleanup** — ensures bundle zip is always deleted even if deploy fails.
- **DLC docs are substantive, not stubs** — requirements have real IDs, user stories have real acceptance criteria, traceability matrix links to real code paths.
